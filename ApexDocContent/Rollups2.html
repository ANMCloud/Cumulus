Rollups Version 2 Engine

NPSP Rollup version 2 functionality is made up of these components:

  • Rollup Custom Metadata Types
  • Opportunity Rollup Batch Classes
  • Opportunity Rollup Triggers and Utilities

Custom Metadata Types used by the Rollups v2 Engine:

  • Filter_Group__mdt
  • Filter_Rule__mdt
  • Rollup__mdt

Classses related to the Custom Metadata Type Filter Engine:

  • CMT_FilterRule
  • CMT_FilterRule_SEL
  • CMT_FilterRuleEval_SVC
  • CMT_FilterRuleUI_SVC
  • CMT_FilterRule_TEST
  • CMT_FilterRuleEval_TEST

Classes related to the Rollups Engine (excluding Batch & Trigger classes):

  • RLLP_Operation
  • RLLP_Query_SEL
  • RLLP_Rollup_SEL
  • RLLP_Rollup_SVC
  • RLLP_RollupAccount_SVC
  • RLLP_VRollupHandler
  • RLLP_Query_SEL_TEST
  • RLLP_Rollup_SEL_TEST
