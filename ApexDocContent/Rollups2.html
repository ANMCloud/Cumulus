Customizable Rollups Engine

NPSP Customizable Rollups functionality is made up of these components:

  • Rollup Custom Metadata Types
  • Opportunity Rollup Batch Classes
  • Opportunity Rollup Triggers and Utilities

Custom Metadata Types used by the Rollups v2 Engine:

  • Filter_Group__mdt
  • Filter_Rule__mdt
  • Rollup__mdt

Classses related to the Custom Metadata Type Filter Engine:

  • CMT_FilterRule
  • CMT_FilterRule_SEL
  • CMT_FilterRuleEval_SVC
  • CMT_FilterRuleUI_SVC
  • CMT_FilterRule_TEST
  • CMT_FilterRuleEval_TEST

Classes related to the Rollups Engine (excluding Batch & Trigger classes):

  • CRLP_FiscalYears
  • CRLP_FiscalYears_TEST
  • CRLP_Operation
  • CRLP_Operation_SVC
  • CRLP_Query_SEL
  • CRLP_Rollup
  • CRLP_Rollup_SEL
  • CRLP_Rollup_SVC
  • CRLP_RollupAccount_SVC
  • CRLP_VRollupHandler
  • CRLP_Query_SEL_TEST
  • CRLP_Rollup_SEL_TEST
